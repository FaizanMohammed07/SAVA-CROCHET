# Sava Crochets

Beautiful handcrafted crochet items made with love, patience, and the finest natural yarns.

## About

Sava Crochets is a boutique crochet brand specializing in handmade amigurumi, blankets, tote bags, hats, flower bouquets, and artisan home decor. Every piece is carefully crafted by hand using premium natural yarns and time-honored techniques.

## Tech Stack

- **React** + **TypeScript** - UI framework
- **Vite** - Build tool
- **Tailwind CSS** - Utility-first CSS
- **shadcn/ui** - Component library
- **Framer Motion** - Animations

## Getting Started

```sh
npm install
npm run dev
```
